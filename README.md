# BrainFuck

A simple [brainfuck](https://en.wikipedia.org/wiki/Brainfuck) interpreter.

```brainfuck
[ hello world! ]

++++++++[>++++[>++>+++>+++>+<<<<-]>+>+>->>+[<]<-]>>.>
---.+++++++..+++.>>.<-.<.+++.------.--------.>>+.>++.
```

## Installation

```bash
npm install -g @yuanchuan/brainfuck
```

## Usage

```bash
$ brainfuck samples/numwarp.bf

1234
       \
      \/\
    /\
     /\
  /\  /
   /
 \ \/
  \
```
